Getting Started¶

Dependencies¶

I assume that the notebook is running on a conda environment. The essential packages are

  1. numpy
  2. scipy
  3. mkl
  4. cython
  5. pandas
  6. matplotlib

These can be installed using the following command:

conda install -c conda-forge numpy scipy mkl cython pandas

The program¶

The essential programs are pre-compiled and are provided by *.so files:

  1. fastrna/utils.so
  2. fastrna/mkl_funcs.so
  3. fastrna/core.so

The libraries can be loaded as if they are the usual *.py files .e.g

import fastrna.utils

The data¶

Three data where each contains the data matrix, gene annotations and cell annotations are attached.

  1. matrix/matrix.npz
  2. matrix/genes.tsv
  3. matrix/cells.tsv

The gene annotations contain the name of the genes. The gene name has two forms, one which is the ensembl name and the common name. The ensembl name is a collection of codes that specifies the gene in the form of EN + Species code + Unique code. The common name is the usual gene name abbreviation, e.g. FTO (Fat mass and obesity-associated protein).

The cell annotations contain the barcode and the batch membership of cells. The barcode are specifiers produced by scRNA-seq machines.

Calculate and visualize QC metrics¶

Three quality control (QC) metrics are employed.

  1. Total number of UMI counts in the data: the sum of all molecule counts in the data (summation over column)
  2. Number of genes expressed in a cell: the number of non-zero genes for each cell
  3. Mitochondrial gene content: number of mitochondrial gene transcripts divided by the number in 1.
In [7]:
# metric 1
qc1 = np.asarray(mtx.sum(axis=0)).ravel()

# metric 2
qc2 = np.asarray((mtx > 0).sum(axis=0)).ravel()

# metric 3
mt_gene = genes.common_name.apply(lambda x: True if 'mt-' in x else False)
qc3 = np.asarray(mtx[mt_gene,:].sum(axis=0) / qc1 * 100).ravel()

fastrna/utils.so and fastrna/mkl_funcs.so contain internal functions to run the program. Therefore, the user should not require the commands inside these files.
